Automating Calibrations (GM)

As a Hardware Engineering Intern at General Motors, I created feature models for vehicle profiles, and Gears Logic using the Gears Product Line Engineering tool, automating calibration values. Feature models include decision trees using implemented logic to make decisions for calibration values for front and rear closures, reducing the number of engineer-driven decisions from 4000+ to sub-13 for each vehicle. This process increased efficiency, and accuracy while driving down time required for setting calibrations.
