Computer Vision-Controlled Guitar Effects System

This project combines a Raspberry Pi (running OpenCV) and STM32 microcontroller to create a tilt-based guitar effects system.

  • OpenCV used to detect guitar tilt in real time
  • Triggered UART commands to control analog switches for signal routing
  • Implemented clean, distortion, and delay effects using PT2399 and analog modules
